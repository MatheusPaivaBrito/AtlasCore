# eventing_api

Platform API for messaging, Kafka governance, event contracts, outbox visibility and event sourcing roadmap.

This service is part of the platform profile. It is grouped by capability, not by vendor/tool.

Planned modules:

- `topics`
- `schemas`
- `event_contracts`
- `outbox`
- `streams`
- `projections`

Provider adapters:

- `kafka`
- `event_store`
