# observability_api

Platform API for observability workflows across Sentry, Grafana and Loki.

This service is part of the platform profile. It is grouped by capability, not by vendor/tool.

Planned modules:

- `incidents`
- `alerts`
- `dashboards`
- `log_queries`
- `releases`

Provider adapters:

- `sentry`
- `grafana`
- `loki`
